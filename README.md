# palworld-python-script

_✨ 适用于palworld windows轮询/计划自动重启服务端 ✨_

## 主要功能

1. 轮询任务/计划任务重启服务端  
2. 重启服务端前通过RCON指令发送关服倒计时  
3. 自定义存档备份时间  

## 使用方法

1. `pyinstaller --onefile task_scheduler.py`打包成exe文件  
2. 配置`config.ini`与`task_scheduler.exe`同目录运行  
具体使用请参考飞书文档
